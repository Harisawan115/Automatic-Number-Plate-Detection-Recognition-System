# 🚗 ANPDRS — Automatic Number Plate Detection & Recognition System

![Python](https://img.shields.io/badge/Python-3.8%2B-blue?logo=python)
![YOLOv8](https://img.shields.io/badge/YOLOv8-Ultralytics-brightgreen)
![EasyOCR](https://img.shields.io/badge/OCR-EasyOCR-orange)
![Streamlit](https://img.shields.io/badge/UI-Streamlit-red)

A deep learning-based system that automatically **detects license plates** from vehicle images and **recognizes the plate number** using OCR — built with YOLOv8 and EasyOCR, deployed via a Streamlit web interface.

## 📦 Dataset

**Name:** Car Plate Detection  
**Source:** Kaggle  
**Link:** [https://www.kaggle.com/datasets/andrewmvd/car-plate-detection](https://www.kaggle.com/datasets/andrewmvd/car-plate-detection)  
**Size:** ~433 images with XML bounding box annotations  
**Class:** `license_plate` (1 class)  
**Split:** 70% Train / 15% Validation / 15% Test

> Download the dataset from Kaggle and place images in `dataset/images/` and XML files in `dataset/annotations/`.

## ⚙️ Setup & Installation

### 1. Clone the repository
```bash
git clone https://github.com/your-username/anpdrs.git
cd anpdrs
```

### 2. Create and activate virtual environment
```bash
python -m venv venv

# Windows
venv\Scripts\activate

# Mac/Linux
source venv/bin/activate
```

### 3. Install PyTorch (GPU)
Check your CUDA version first:
```bash
nvidia-smi
```
Then install:
```bash
# CUDA 12.1
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121

# CUDA 11.8
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
```

### 4. Install remaining dependencies
```bash
pip install -r requirements.txt
```

---

## 🚀 How to Run

### Step 1 — Convert XML annotations to YOLO format
```bash
python convert_annotations.py
```

### Step 2 — Train YOLOv8
```bash
python train_yolo.py
```
Training takes ~20–30 minutes on an RTX 2050 4GB GPU.  
Best weights are saved at: `runs/detect/yolov8_plates/weights/best.pt`

### Step 3 — Launch Streamlit App
```bash
streamlit run app_fixed.py
```
Open your browser at: **http://localhost:8501**
